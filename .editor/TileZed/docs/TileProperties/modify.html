You may add commands to the TileProperties.txt file that will modify the properties in a .tiles file as it is read in.  This can be useful if you want to add, rename, or remove properties or change the value of properties on many tiles without having to edit each one.

To add a command, place one or more blocks named "modify" in TileProperties.txt.
modify
{
    command = arg
    command = arg arg
    command = arg arg arg arg
    ....
}
modify
{
    command = arg
}
....
Each "modify" block has a list of one or more commands that will be executed in order on the set of properties read in for every tile in a .tiles file.  All of the modify blocks are processed on every tile's properties in the order the modify blocks appear in TileProperties.txt.

Any of the arguments to a command may be enclosed in double-quotes "...".  If you wanted to match an empty value, just write "".

Command: match

match = property
match = property value

The match command looks for either a property or a property/value pair.  If no matching property is found, then none of the other commands after the match command in the current modify block will be executed.

Command: reject

reject = property
reject = property value

The reject command has the opposite effect of the match command.  If a tile does have a property set, then none of the commands after the reject command in the current modify block will be executed.

Command: remove

remove = property
remove = property value

The remove command removes the named property from the set of properties on a tile.  If no value is given, then the named property is removed regardless of the value.

Command: rename

rename = old_name new_name

The rename command changes the name of a property called old_name to new_name.  If the property called old_name isn't found on a tile, this command has no effect.

Command: replace

replace = property1 value1 property2 value2

The replace command removes the property1/value1 pair and adds the property2/value2 pair.  If the property1/value1 pair is not found, then this command has no effect.

Command: set

set = property value

The set command sets the value of a property.  This command sets the property/value even if the named property is not already set on a tile.
