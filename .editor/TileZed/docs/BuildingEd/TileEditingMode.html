Tile Mode

Use Tile Mode to draw tiles by hand like you would in TileZed.  In general, you should always try to use the tools in Ortho Mode or Iso Mode to create your buildings.  Any tiles you paint in Tile Mode will be erased if you rotate or flip a building.

Available Tile Layers

BuildingEd always creates certain tile layers that are needed by rooms and objects.  Additional layers may be added for use in Tile Mode by editing the TMXConfig.txt file as explained here.  Any layers added to TMXConfig.txt will be available for new and existing buildings.  Adding new layers won't hurt existing buildings.  If you remove a layer from TMXConfig.txt, you won't be able to draw or erase tiles in those layers.  You will get an error starting TileZed if you remove any of the layers required by BuildingEd.

The Furniture Dock

Strangely enough, there is a Furniture Dock in Tile Mode.  The Furniture Dock allows you to paint all the tiles that make up a piece of furniture into the current tile layer.  Normally, you would just place a furniture object in Ortho/Iso Mode.
